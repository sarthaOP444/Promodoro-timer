Focus — Pomodoro Timer
A sleek, iOS-inspired Pomodoro timer web app to help you stay productive with focused work sessions and scheduled breaks.
🔗 Live Demo: promodorotimerapp.netlify.app
Features
⏱️ Focus / Short Break / Long Break modes with a smooth animated progress ring
✅ Task management — add, complete, and track tasks tied to your sessions
📊 Statistics dashboard — daily sessions, focus time, streaks, and a 7-day activity chart
⚙️ Customizable settings — adjustable durations, session cycles, auto-start options, sound/haptic feedback, and quick presets
🌗 Light & dark themes
📱 Fully responsive — polished for mobile, tablet, and desktop with a native app-like feel
Tech Stack
Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.
Usage
Simply open the app and start a focus session. Customize your durations and preferences in Settings, add tasks to track what you're working on, and check your Statistics to monitor your productivity over time.
